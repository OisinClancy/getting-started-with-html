Lesson 1

World Wide Web - Consists of a massive system of interconnected computers that communicate with each other through use of HTML Documents.

HTML - Hyper Text Markup Language

Tags - Different elements are found inside brackets such as < > or closing brackets < / >

Elements - These are the properties that are found inside tags such as 'b' for bold and 'em' for italic

Attribute - These are attached to elements to help such as 'src' for an image or 'href' for a link

Inline/Block - Two different types of elements. Block elements form 'invisivble boxes' around the content they surround.
